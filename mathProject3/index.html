Monty Hall problem

There are 3 doors behind only 1 there is prize. A contestant chooses one of the 3 doors. Then the game host opens one of the other two doors behind which there is no prize. The contestant then gets to either keep the door they chose, or switch to the other door.

What option will have the highest win rate?

How to simulate:

Choose how many times you want to simulate the game from the dropdown menu, then click simulate. When the simulation is done, two graphs will be displaying the win loss ratio

Game explanation

There is clearly a trend here. By running the simulation 1 million, 100 thousand, 10 thousand, a thousand or 100 times we get a huge dataset. With this data set we see that almost every time there is 2/3 of winning the game by switching, but when we stay with the original pick we see that there is only 1/3 chance og winning the game.

For some it might come as a suprise that it's not a 50/50 chance of winning the game. When monty opens a door, instead of having a 1/3 chance of winning, you as the player get another chance, only this time instead of 1/3 chance you have a 1/2 chance in getting the right door.
